Project: Retail Price Intelligence Engine

A cloud-native automated monitoring system designed to track pricing and inventory changes across e-commerce platforms. Built with Python, Playwright, and Pandas, running on Azure VMs via Docker.

Key Features:

Headless Scraping: Utilizes Playwright for dynamic content rendering.

Event-Driven Alerts: Integrated with mobile push notifications for real-time price drop alerts.

Data Pipeline: Cleans and structures data using Pandas for analysis.

CI/CD: Automated builds using GitHub Actions.

Tech Stack: Python 3.11, Playwright, Docker, Azure DevOps.
